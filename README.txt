
-This is a simple code to shut the computer down in case 2 hours have elapsed once.
-Create the folder "C:\\Anti Younger Brother".
-It creates a file called "test" in "C:\\Anti Younger Brother" and writes number 1.
-To use it for 2 hours again erase the number and save the file.
-To have it running when the computer starts, build the program and paste it in the windows startup folder.
-To increase or decrease the time change the number of seconds from "7200" to the desired number.

pt-br

-Este é um código simples para desligar o computador no caso de se passarem 2 horas.
-Crie a pasta "C:\\Anti Younger Brother".
-Cria um arquivo chamado "test" em "C:\\Anti Younger Brother" e escreve o número 1.
-Para usar por duas horas again apague o numero e salve o arquivo.
-Para que comece a contar quando o computador inicia, compile o programa e cole na pasta inicializar do windows.
-Para aumentar ou diminuir o tempo mude o numero de segundos de "7200" para o desejado.
